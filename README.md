Weather App 
A Python-based weather application that retrieves real-time weather data using a public API. Users can search by city name or ZIP code to view current temperature, humidity, wind speed, and weather conditions. The project includes robust error handling, unit conversion (Celsius/Fahrenheit), and optional GUI support with weather icons, hourly and daily forecasts, and a clean, user-friendly interface.

Features

Real-time weather data retrieval using a public weather API
Search weather by city name or ZIP code
Displays temperature, humidity, wind speed, and weather conditions
Supports temperature unit conversion (Celsius/Fahrenheit)
User-friendly command-line interface (CLI)
Optional graphical user interface (GUI) with weather icons
Hourly and daily weather forecasts (GUI version)
Input validation and robust error handling
Clean and beginner-friendly Python code structure
